# X に自動投稿するための bot を作成する

## Working Agreement for Agents

- 設計と実装は常にシンプルにする
- DRY原則。再利用を心がける
- 曖昧な指示や不明点があれば必ず確認する。絶対に憶測で実装をしない
- 可能な限りテストコードを用意する。しかし、テストコードを用意するために実装を複雑にしない
- 仕様変更時は `README.md` と `CHANGELOG.md` の整合性を確認する
- 会話は日本語で行う

## アーキテクチャ

golang で実装する
daemon として実装して、任意の投稿仕様を追加実装しやすいように心がける
完成後は Ubuntu Linux に deploy して、systemctl にて制御する
